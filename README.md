# Bootstrap Modern Business Projesi

## 📌 Proje Hakkında

Bu proje, HTML, CSS, Bootstrap 5 ve Font Awesome kullanılarak hazırlanmış bir frontend geliştirme çalışmasıdır.

Amaç, verilen örnek bir modern business (iş/şirket) tasarımını inceleyerek **Bootstrap bileşenleri ile sıfırdan yeniden oluşturmaktır**.

Proje geliştirilirken DevTools kullanılarak sayfa yapısı analiz edilmiş, Bootstrap bileşenleri öğrenilerek uygulanmıştır.

---

## 🎯 Projenin Amacı

Bu proje ile amaçlananlar:

- Bootstrap Navbar yapısını öğrenmek
- Responsive (mobil uyumlu) tasarım geliştirmek
- Bootstrap grid sistemini kullanmak
- Card, button ve ikon yapılarıyla çalışmak
- Çok sayfalı (multi-page) site mantığını anlamak
- Gerçek bir template’i analiz ederek yeniden oluşturmak

---

## 🧩 Sayfalar

Bu projede sadece aşağıdaki sayfalar bulunmaktadır:

- Home (Ana Sayfa)
- About (Hakkımızda)
- Contact (İletişim)

Navbar üzerinden bu sayfalar arasında geçiş yapılmaktadır.

## 📱 Responsive Yapı

- Site Bootstrap breakpoint sistemine göre responsive olarak tasarlanmıştır
- Mobil ve masaüstü uyumludur
- Navbar mobil cihazlarda collapsible (açılır kapanır) yapıya sahiptir
